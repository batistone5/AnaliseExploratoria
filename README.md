# AnaliseExploratoria
Atividade de Prática Estatística sobre o GitHub

Análise dos Gráficos criados no Script 1.

Histograma das idades:

  É possível ver uma distribuição que se assemelha a uma distribuição normal, com
a maioria dos indivíduos estando na faixa dos 46 aos 60 anos de idade, e a minoria
tendo mais de 88 anos.


Boxplot do tempo preso:

 Pelo boxplot podemos perceber que os dados possuem uma baixa variação, que a mediana
se encontra próximo dos 60 meses, e existe um outlier próximo dos 90 meses.


Boxplot do tempo preso por nível de escolaridade:

  Comparando os 3 boxplots, não percebemos uma diferença significatia entre eles a primeira
vista, apenas uma variação ligeiramente maior nos indivíduos com nivel médio de escolaridade.


Gráfico de barras para a reincidência de prisão:

  Podemos ver que o número de indivíduos não reincidentes é bem maior que o númeor de
reincidente, talvez 4 vezes maior.
